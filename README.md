AERIS

AERIS is a mobile-first social app combining Feed, Chat, Moments (Stories), and AI-enhanced conversations (ERIZ AI). This repository contains a Flutter mobile client (mobile/) and a Node.js backend (backend/) in a single monorepo. The app uses a bottom navigation with Home, Chat, and Account. ERIZ AI is pinned at the top of Chat. Moments (Stories) live in Home as a horizontal row.
